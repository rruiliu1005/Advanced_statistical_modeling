One-dimensional Kernel Density Estimation - Kernel Choices

Let \(X_1,...,X_n\) be a random sample from a population with continuous pdf f. The kernel density estimator of f is: \[\widehat{f}_h(x) = \frac{1}{nh}\sum_{i=1}^nK\left(\frac{x-X_i}{h}\right)\] where \(K(\cdot)\) is the kernel function and h is the bandwidth.

One-dimensional Kernel Density Estimation - Bandwidth Selection

Method 1: Selected optimal bandwidths based on AMISE

The optimal bandwidths for kernel density estimators can be selected based on asymptotic mean integrated square error (AMISE) where
\[AMISE\{\widehat{f}_h(x)\} = \frac{5}{4}n^{-\frac{4}{5}}\{R(f'')\}^{\frac{2}{5}}\{R(K)\}^{\frac{4}{5}}\{\mu_2(K)\}^{\frac{2}{5}}\] the optimal bandwidth that minimized the AMISE is: \[h_{opt} = \left[\frac{R(K)}{n\color{#AA3A39}{R(f'')}{\mu_2(K)}^2}\right]^{\frac{1}{5}}\] Problem: we need to estimate \(f''\)

Solution: Rule of Thumb
Estimate \(R(f'')\) parametrically by assuming that f is close to a Gaussian density function.

Gaussian: \(\widehat{h}_{rot} = \left(\frac{1}{4\pi}\right)^{\frac{1}{10}}\left(\frac{3}{8\sqrt{\pi}}\right)^{-\frac{1}{5}}\widehat{\sigma} n^{-\frac{1}{5}}\approx 1.06\widehat{\sigma} n^{-\frac{1}{5}}\)
Epanechnikov: \(\widehat{h}_{rot} = \left(\frac{5\widehat{\sigma}^5}{n}\right)\approx 2.34\widehat{\sigma} n^{-\frac{1}{5}}\)
Uniform: \(\widehat{h}_{rot} = \left(\frac{3\widehat{\sigma}^5}{n}\right) \approx 2.77\widehat{\sigma} n^{-\frac{1}{5}}\)

Method 2: Select optimal bandwidths based on ISE

The optimal bandwidths for kernel density estimators can be selected based on the integrated squared error (ISE) where
\[ISE(\widehat{f}_h) = \int_{-\infty}^{\infty} \left\{\widehat{f}_h(x) - f(x)\right\}^2dx = \int_{-\infty}^{\infty} \left\{\widehat{f}_{h}(x)\right\}^2dx - \color{#AA3A39}{2\int_{-\infty}^{\infty} \widehat{f}_h(x)f(x)dx} + \int_{-\infty}^{\infty} \left\{f(x)\right\}^2dx\]

Problem:
\(-2\int_{-\infty}^{\infty} \widehat{f}_h(x)f(x)dx = -2E\{g(X)\}|_{g=\widehat{f}_{h}}\) depends on \(h\) and involves unknown quantity f. 

Solution: Least Squares Cross-validation
Find \(h_{LSCV}\) that minimizes: \[LSCV(h) = \int_{-\infty}^{\infty} \left\{\widehat{f}_{h}(x)\right\}^2dx - \frac{2}{n} \sum_{i=1}^n \widehat{f}_{h,-i}(X_i) \\ =\frac{1}{n^2h}\sum_{i=1}^n\sum_{j=1}^nK_{(2)}\left(\frac{X_i - X_j}{h}\right) - \frac{2}{n(n-1)h}\sum_{i=1}^n\sum_{j\neq i}K\left(\frac{X_i - X_j}{h}\right)\] \(h_{LSCV}\) is asymptotically optimal

One-dimensional Kernel Density Estimation - Boundary Bias

Problem: When we use the kernel density estimator \(\widehat{f}_{h}{(x)}\) to estimate \(f(x)\), the support set of the estimator can be different from that of \(f(x)\).

Specifically, the bias of our estimator \(\widehat{f}_{h}{(x)}\) at \(x = ph\) with \(0 \leq p < 1\) is: \[Bias\left\{\widehat{f}_{h}{(x)}\right\} = \{a_0(p) - 1\}f(x) - ha_1(p)f'(x)+\frac{1}{2}h^2a_2(p)f''(x)+o(h^2)\] where \(a_l(p) = \int_{-\infty}^pu^lK(u)du\)

Solution 1: Normalization We normalize \(\widehat{f}_{h}{(x)}\) by dividing by \(a_0(p)\) so that: \[\tilde{f_h}(x) = \frac{\widehat{f}_{h}(x)}{a_0(p)}\] and \[Bias\left\{\tilde{f_h}{(x)}\right\} = - h\frac{a_1(p)}{a_0(p)}f'(x)+\frac{1}{2}h^2\frac{a_2(p)}{a_0(p)}f''(x)+o(h^2) \rightarrow 0\] Note:
The bias has rate \(h\) instead of \(h^2\), meaning that the convergence at boundary is much slower.

Solution 2: Boundary Kernels Suppose we have two kernel functions \(L(\cdot)\) and \(K(\cdot)\), we know that \[ E\{c_1(p)\widehat{f}_{K,h}{(x)}\} = c_1(p)a_0(p)f(x) - ha_1(p)c_1(p)f'(x)+O(h^2) \\ E\{a_1(p)\widehat{f}_{L,h}{(x)}\} = a_1(p)c_0(p)f(x) - ha_1(p)c_1(p)f'(x)+O(h^2) \\ \] we define a new kernel \(B(\cdot)\) at the boundary: \[B(u) = \frac{c_1(p)K(u) - a_1(p)L(u)}{c_1(p)a_0(p) - a_1(p)c_0(p)}\] The new estimator \(\widehat{f}_{B,h}{(x)}\) satisfies: \[E{\widehat{f}_{B,h}{(x)}} = f(x) + O(h^2)\]

Solution 3: Log - Transformation We estimate a kernel density estimator \(\widehat{g}(\cdot)\) based on the log transformed data and the estimator for the original variable y is \[\widehat{f}(x) = \frac{\widehat{g}(log(x))}{x}\] Note: If x is near 0 at the boundary, then the result will be very unstable.

Solution 4: Reflected Method We use a new estimator \[\widehat{f}(x) = \frac{1}{nh}\left[ K\left( \frac{x+X_i}{h}\right) + K\left( \frac{x-X_i}{h}\right)\right]\] to estimate the density at the boundary.
Essentially, we are reflecting the kernel at the boundary.

Two-dimensional Kernel Density Estimation

If \(X \in \mathbb{R}^2\), then the two dementional kernel density estimator of \(f(x)\) is:

\[\widehat{f}_h(x) \ = \frac{1}{n}\sum_{i=1}^n \frac{1}{h_1 h_2}K\left( \frac{x-X_1}{h_1}, \frac{x-X_2}{h_12}\right)\]

More generally, we use a non-singular matrix \(H\) as the bandwidth matrix, then

\[ \widehat{f}_h(x) \ = \frac{1}{n}\sum_{i=1}^n \frac{1}{det(\mathbf{H})}\mathscr{K}\{\mathbf{H}^{-1}(\mathbf{x} - \mathbf{X_i})\} \]

Topic 3: Kernel Regression

Scaling Data

For our regression analysis, we scale the data by standardizing the data so that the distance calculation are consistent across the covariates.

Local Constant and Local Linear Kernel Regression

We model the data non-parametrically as: \[Y_i = m(\mathbf{X_i}) + \epsilon_i\]

In a local constant estimator , the estimator \(\widehat{m}_{h}(x)\) is defined as:

\[\widehat{m}_{h}(x) = \frac{\sum_{i=1}^nK\left(\frac{x-X_i}{h}\right)Y_i}{K\left(\frac{x-X_i}{h}\right)}\]

In a local linear estimator , the estimator \(\widehat{m}_{h}(x)\) is defined as:

\[\widehat{m}_{1,h}(x) = \frac{S_{2,h}(x)T_{0,h}(x) - S_{1,h}(x)T_{1,h}(x)}{S_{0,h}(x)S_{2,h}(x)- S^2_{1,h}(x)} \qquad \]

where \[\widehat{\beta}(x)= \begin{pmatrix} S_{0,h}(x) & S_{1,h}(x) \\ S_{1,h}(x) & S_{2,h}(x) \end{pmatrix}^{-1} \begin{pmatrix} T_{0,h}(x) \\ T_{1,h}(x) \end{pmatrix} \qquad S_{j,h}(x) = \sum_{i=1}^n K_h(x-X_i)(X_i-x)^j \qquad T_{j,h}(x) = \sum_{i=1}^n K_h(x-X_i)(X_i-x)^jY_i \]

Topic 4: Regression Splines

Regression Spline

The regression spline estimator of \(m(x)\) is defined as: \[\widehat{m}_L(x) = B^T(x)\widehat{\beta} = B^T(x)(B^TB)^{-1}B^TY\] where \[\mathbf{B}(x) = \begin{pmatrix} B_{-d, d+1}(x) \\ \vdots \\ B_{L, d+1}(x)\end{pmatrix} \qquad \mathbf{B} = \begin{pmatrix} \mathbf{B}(X_1) \\ \vdots \\ \mathbf{B}(X_n)\end{pmatrix} \qquad \mathbf{\beta} = \begin{pmatrix} \beta_{-d} \\ \beta_{-d+1} \\ \vdots \\ \beta_L\end{pmatrix} \qquad \mathbf{Y} = \begin{pmatrix} Y_1 \\ Y_2 \\ \vdots \\ Y_n\end{pmatrix} \] d is the degree of the spline function, L is the number of knots used in the estimation and \(\mathbf{\beta}\) are the minimizers of s \[\sum_{i=1}^n\left\{ Y_i - \sum_{l=-d}^L\beta_l B_{l,d+1}(X_i)\right\}^2\]

Regression Spline

We use a regression spline to estimate the relationship between the VIX index and Oil prices
We sample 75% of the data without replacement as our training set and the rest 25% are used as testing set.
The optimal number of knots is selected using leave-one-out cross validation defined as: \[CV(L) = \frac{1}{n}\sum_{i=1}^n(\{\mathbf{I}_n - diag(\Omega_n)\}_{ii}\{Y_i - \widehat{m}_L\}^2w(X_i)\] with \(w(\cdot)\) as the indicator function checking if \(X_i\) falls within the middle 90% of the data.

Penalized Regression Spline

The penalized regression spline estimator of \(m(x)\) is defined as: \[\widehat{m}_{\lambda}(x) = \mathbf{B}(x)(\mathbf{B}^T\mathbf{B}+\lambda \mathbf{D}_q)^{-1}\mathbf{B}^TY\] where \[D_q = \int_a^b\mathbf{B}^{(q)}(x)\{\mathbf{B}^{(q)}(x)\}^Tdx\] The penalized spline regression estimator minimizes the following equation with respect to \(s \in S_d(t)\) \[\sum_{i=1}^n\{Y_i-s(X_i)\}^2 + \lambda \int_a^b\{s^{(q)}(x)\}^2dx\] The penalized regression spline balances the trade-off between fitting the observed data closely and maintaining smoothness by penalizing excessive curvature.

Tensor Product Spline

The tensor product regression spline estimator of \(m(x)\) is defined as:

\[\widehat{m}(\mathbf{x}) = \{B_{[r]}(x)\}^T\hat{c} = \{B_{[r]}(\mathbf{x})\}^T(\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{Y}\] where \[B_{[r]}(\mathbf{x}) = \otimes_{j=1}^rB_j(x_j) \qquad \mathbf{X} = \begin{pmatrix} B_{[r]}(\mathbf{X_1}) \\ B_{[r]}(\mathbf{X_2}) \\ \vdots \\ B_{[r]}(\mathbf{X_n}) \end{pmatrix}\]
